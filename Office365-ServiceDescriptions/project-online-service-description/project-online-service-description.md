# <a name="project-online-service-description"></a>Project Online 서비스 설명

Project Online은 프로젝트 PPM (포트폴리오 관리) 및 일상 업무에 대 한 유연한 온라인 솔루션입니다. Office 365를 통해 배달, Project Online 강력한 프로젝트 관리 기능을 제공 계획, 우선순위를 지정 하 고, 프로젝트 및 프로젝트 포트폴리오 투자를 관리 하기 위한-거의 어디에서 든 지 거의 모든 장치에 있습니다. Project Online 관리자, 포트폴리오 관리자 및 뷰어, 프로젝트 및 자원 관리자 및 팀 리더 및 구성원으로 사용할 수 있습니다. 
  
Project Online은 Project Online Essentials, Project Online Professional 및 Project Online Premium의 3가지 계획으로 사용할 수 있습니다.
  
계획 간 기능을 비교하려면 [Project Online 계획 비교](https://products.office.com/en-us/project/compare-microsoft-project-management-software)를 참조하세요.
  
Office 365 서비스 설명의 페이지를 내보내고 저장하며 인쇄할 수 있습니다. [여러 페이지 내보내기](https://go.microsoft.com/fwlink/?LinkId=403349) 방법을 알아보세요.
  
## <a name="project-online-boundaries-and-limits"></a>Project Online 경계 및 제한

Project Online 서비스가 일부 제한됩니다. 자세한 내용은 [Project Online 소프트웨어 경계 및 제한](https://support.office.com/en-us/article/Project-Online-software-boundaries-and-limits-5A09DBCE-1E68-4A7B-B099-D5F1B21BA489)을 참조하세요.
  
## <a name="data-backup"></a>데이터 백업

 Project Online은 SharePoint Online 위에 구축되며 SharePoint와 같은 데이터 백업 정책을 따릅니다. 자세한 내용은 [SharePoint Online 서비스 설명](https://technet.microsoft.com/en-us/library/sharepoint-online-service-description.aspx)에서 데이터 백업 섹션을 참조하세요.
  
## <a name="use-of-sharepoint-online"></a>SharePoint Online 사용

프로젝트 온라인 서비스에는 SharePoint Online, Project Online 서비스의 일부로 귀하에 대 한 프로 비전 된 수는 사용을 해야 합니다. Project Online Professional 또는 프리미엄와 함께 제공 되는 SharePoint Online 기능에 대 한 권한을 저장 하 고 프로젝트 온라인 지원 하기 위한 데이터 액세스 (영문)으로 제한 됩니다.
  
## <a name="licensing-considerations"></a>라이선스 고려 사항

### <a name="internal-users"></a>내부 사용자

Project Online 사이트에서 모든 상호작용에는 Project Online 라이선스가 필요 합니다.
  
- Project Online Essentials는 최소 요구 사항입니다.
    
- 고급 사용 사례에 대 한 사용자에는 적절 한 라이선스를 보유 해야 합니다. 프로젝트 관리자는 Project Online Professional 라이선스가 최소한 필요 합니다. 포트폴리오 관리자에는 프로젝트 온라인 프리미엄 라이선스가 필요 합니다.
    
> [!IMPORTANT]
>  Project Online Essentials은 Project Online Professional 또는 프로젝트 온라인 프리미엄 고객을 위한 팀 구성원 *추가* 라이선스입니다.<br/>마지막 Project Online Professional 또는 프로젝트 온라인 프리미엄 라이선스 만료 되 면 Project Web App 인스턴스 120 일 후 삭제 됩니다.<br/>Project Online 평가판 구독에 대 한 Project Web App 인스턴스 30 일 평가 기간이 프로그램을 종료 한 후 삭제 됩니다. 
  
### <a name="external-users"></a>외부 사용자

Project Online 외부 사용자에 대 한 라이선스에 내부 사용자에 대 한 것과 같은 정책을 다음과 같습니다. Project Online 사이트에서 모든 상호작용에는 Project Online 라이선스가 필요 합니다.
