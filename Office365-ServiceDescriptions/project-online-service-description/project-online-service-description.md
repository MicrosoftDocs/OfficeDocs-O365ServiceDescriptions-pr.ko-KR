# <a name="microsoft-project-service-description"></a>Microsoft Project 서비스 설명

## <a name="overview"></a>개요

Microsoft Project 프로젝트 및 작업 관리에 대한 조직의 요구를 충족하는 데 도움이 되는 다음 응용 프로그램을 제공합니다.

- 웹용 Project
- Project Online 
- Project Online 데스크톱 클라이언트

이 문서에서는 각 Project 구독에 제공되는 응용 프로그램과 각 응용 프로그램에서 제공하는 기능을 이해하는 데 도움이 됩니다.

### <a name="project-for-the-web"></a>웹용 Project

웹의 Project 클라우드 기반 작업 및 프로젝트 관리를 위한 Microsoft의 가장 최근 제품입니다. 웹용 Project 대부분의 요구와 역할을 충족하는 간단하고 강력한 작업 관리 기능을 제공합니다. Project 관리자와 팀 구성원은 웹에 Project 사용하여 모든 규모의 작업을 계획하고 관리할 수 있습니다.

웹용 Project Microsoft 전원 플랫폼에 기반을 두고 있습니다. 파워 플랫폼은 PowerApps, Power Automate, Power BI 및 Microsoft 데이터 버스로 구성됩니다. 웹 데이터에 대한 Project Microsoft Dataverse에 저장됩니다. 

웹용 Project Project 플랜 1, Project 플랜 3 및 Project 플랜 5 세 가지 구독을 통해 사용할 수 있습니다. [웹 기능에 대해 Project](#project-for-the-web-features) 나열된 기능은 웹 데이터 저장소에 대한 Project 데이터와만 상호 작용합니다.

### <a name="project-online"></a>Project Online
 
Project Online Project 포트폴리오 관리(PPM) 및 일상 업무를 위한 유연한 온라인 솔루션입니다. Project Online 거의 모든 장치에서 프로젝트 및 프로젝트 포트폴리오 투자를 계획, 우선 순위 지정 및 관리하기 위한 강력한 프로젝트 관리 기능을 &mdash; 제공합니다. Project Online은 관리자, 포트폴리오 관리자 및 열람자, 프로젝트 및 리소스 관리자, 팀 책임자 및 팀원들이 사용할 수 있습니다.

Project Online SharePoint 플랫폼에 구축되어 있으며 SharePoint 데이터 저장소에 데이터를 저장합니다.

Project Online Project 플랜 3 Project 플랜 5 두 개의 구독을 통해 사용할 수 있습니다. [Project Online 기능에](#project-online-features) 나열된 기능은 Project Online SharePoint 데이터 저장소의 데이터와만 상호 작용합니다.

Project Online 대한 팀 멤버 기능은 Project Online Essentials 또는 Project 플랜 1 구독을 통해 사용할 수 있습니다. Project Online Essentials Project 플랜 3 또는 Project 플랜 5 구독을 가진 고객을 위한 팀 멤버 *추가 기능* 구독입니다. Project Online Essentials 및 Project 플랜 1 구독은 팀 구성원에게 Project Online 관리되는 프로젝트에 대한 작업 및 시간 업데이트를 수행할 수 있는 웹 인터페이스를 제공합니다.

Project Online Essentials 또는 Project 플랜 1 구독이 있는 팀 멤버는 다음 Project Online 기능을 수행할 수 있습니다.

- 웹 기반 인터페이스 사용
- 업데이트 작업, 문제점 및 위험
- 작업표 제출
- 문서를 공유하고 Microsoft Teams 또는 비즈니스용 Skype 사용자와 공동 작업

Project Online Essentials 웹에 대한 Project 포함하지 않습니다. Project 플랜 1 웹 및 Project Online Essentials 대한 Project 모두 포함합니다.

### <a name="project-online-desktop-client"></a>Project Online 데스크톱 클라이언트

Microsoft Project Online 데스크톱 클라이언트는 자동화된 스케줄링, 프로젝트 리소스 관리 및 기본 제공 보고를 특징으로 하는 프로젝트 관리 프로그램입니다. 데스크톱 클라이언트를 독립 실행형 응용 프로그램으로 사용하거나 Project Online 연결할 수 Project Online 있습니다. Project Online 데스크톱 클라이언트는 Project Professional 데스크톱 클라이언트의 구독 버전이며 Project 플랜 3 및 Project 플랜 5 구독에서 사용할 수 있습니다.

### <a name="using-project-for-the-web-and-project-online-together"></a>웹에 Project 사용하고 함께 Project Online

웹 및 Project Online 대한 Project 별도의 응용 프로그램이지만 나란히 사용할 수 있습니다. 홈Project두 응용 프로그램에서 모든 프로젝트를 볼 수 있습니다. Project 로드맵 기능을 사용하여 두 응용 프로그램의 프로젝트를 포함하는 로드맵을 빌드할 수 있습니다. 또한 웹 Power BI 콘텐츠 팩의 Power BI 및 Project 사용하여 두 응용 프로그램에서 프로젝트 및 리소스에 대한 통찰력을 얻을 수 있습니다. <sup>3</sup>

이 다이어그램은 응용 프로그램이 나란히 작동하는 방법을 보여 주습니다.

![웹과 Project Online 함께 Project 보여주는 다이어그램](../media/project-online-applications.png)

앞으로 Microsoft는 웹 혁신을 위한 Project 집중할 것입니다. 그러나 주요 성능 및 보안 개선을 계속 받을 수 있으므로 Project Online 안심하고 계속 사용할 수 있습니다.

<sup>8</sup> Project 플랜 3 또는 Project 플랜 5 구독을 가진 사용자는 리소스 참여 요청을 제출할 수 있습니다. 리소스 참여 요청을 검토, 이행 또는 승인할 수 없습니다.</br>
<sup>9</sup> Project 플랜 5 구독을 한 사용자는 리소스 참여 요청을 검토, 이행 및 승인할 수 있습니다.</br>

## <a name="service-considerations"></a>서비스 고려 사항

> [!NOTE]
> 웹용 Project 현재 GCC, GCC 높음 및 DoD에 사용할 수 없습니다. 웹에 대한 Project 제공하기 위해 노력하고 있지만 현재 는 확고한 기간을 공유할 수 없습니다.

### <a name="licensing-considerations"></a>라이선스 고려 사항

- Project Online 사이트의 상호 작용에는 테넌트 내에서 Project 플랜 3 또는 Project 플랜 5 구독이 필요합니다.
- 마지막 Project 플랜 1, Project 플랜 3 또는 Project 플랜 5 구독이 만료되면 Microsoft Dataverse에 종속된 활성 구독이 없을 때까지 웹 인스턴스에 대한 Project 자동으로 삭제되지 않습니다.
- 마지막 Project 플랜 3 또는 Project 플랜 5 구독이 만료되면 Project Online 인스턴스는 120일 후에 삭제됩니다.
- Project Online 평가판 구독의 경우 평가판 기간이 종료된 후 30일 후에 평가판 인스턴스가 삭제됩니다.
- 웹 평가판 구독에 대한 Project 경우 Microsoft Dataverse에 종속된 활성 구독이 없을 때까지 평가판 인스턴스가 삭제되지 않습니다.

### <a name="use-of-sharepoint-online"></a>SharePoint Online 사용

Project Online Project Online 일환으로 프로비전된 온라인 SharePoint 사용을 요구합니다. Project 플랜 3 또는 Project 플랜 5 구독과 함께 제공되는 SharePoint 온라인 기능에 대한 권리는 Project Online 지원하기 위해 데이터를 저장하고 액세스하는 것으로 제한됩니다.

### <a name="project-roadmap-and-power-automate"></a>Project 로드맵 및 Power Automate

Project 로드맵에는 Project 구독의 일부로 프로비전된 Power Automate 사용해야 합니다. Power Automate 기능에 대한 권한은 Project 로드맵에 필요한 Power Automate 기능으로 제한됩니다. Project 로드맵에 필요한 Power Automate 기능은 Microsoft 365 관리 센터의 **앱** 섹션에서 Flow 서비스 요금제와 **Project 위한 데이터 통합으로** 나타납니다.

웹에 대한 Project 데이터를 저장하기 위해 Microsoft Dataverse를 사용해야 합니다. Microsoft 데이터버스 데이터베이스는 Project 구독의 일부로 프로비전됩니다. Microsoft Dataverse 기능에 대한 권한은 웹에 대한 Project 지원하기 위해 데이터를 저장하고 액세스하는 것으로 제한됩니다. Project 필요한 Microsoft Dataverse 기능은 Microsoft 365 관리 센터의 **앱** 섹션에서 서비스 Project **위한 공통 데이터 서비스로** 나타납니다.
