# <a name="powerpoint-online"></a>PowerPoint Online

powerpoint Online (이전의 powerpoint Web App)을 사용 하면 Microsoft powerpoint 환경을 웹 브라우저로 확장 하 여 프레젠테이션이 저장 된 웹 사이트에서 프레젠테이션을 직접 사용할 수 있습니다. Office Online을 사용하는 Microsoft Office 365 고객은 어디에서나 파일을 보고, 만들고, 편집할 수 있습니다.
  
PowerPoint 편집기는 사용자가 충실도를 잃지 않고 문서 작업을 하는 데 사용할 수 있는 브라우저 기반 편집 공간을 만드는 웹 프런트 엔드 구성 요소입니다.
  
powerpoint Online 도구 모음에서 **powerpoint에서 열기** 단추를 클릭 하면 프레젠테이션이 powerpoint 데스크톱 앱에서 열립니다 (Microsoft PowerPoint 2010 이상이 컴퓨터에 설치 되어 있는 경우). 
  
[비즈니스용 Office 365를 사용하여 PC에 Office 다운로드 및 설치](https://support.office.com/en-us/article/Download-and-install-Office-using-Office-365-for-business-on-a-PC-72977511-dfd1-4d8b-856f-405cfb76839c) 방법을 알아보세요.
  
## <a name="alignment-bullets-numbers"></a>맞춤, 글머리 기호, 번호
<a name="bkmk_AlignmentBulletsNumbers"> </a>

PowerPoint Online에서는 단락 맞춤을 변경하고, 글머리 기호를 적용하거나 번호를 지정하며, 글머리 기호 또는 번호가 지정된 텍스트 수준을 변경할 수 있습니다. 자세한 내용은 [브라우저와 PowerPoint 데스크톱 앱 간의 프레젠테이션 사용 차이점](https://go.microsoft.com/fwlink/?LinkId=272763)을 참조하세요.
  
## <a name="apply-basic-transitions-and-animations"></a>기본 전환 및 애니메이션 적용
<a name="bkmk_Applybasictransitions"> </a>

애니메이션 및 전환 효과를 갤러리에서 선택합니다. PowerPoint Online은 8가지 전환 효과와 37가지 애니메이션을 지원합니다. PowerPoint Online에서 지원되지 않는 추가 애니메이션과 전환은 프레젠테이션에 유지되고 슬라이드 쇼에 표시되지만, PowerPoint Online에서 수정할 수는 없습니다. 자세한 내용은 [브라우저와 PowerPoint 데스크톱 앱 간의 프레젠테이션 사용 차이점](https://go.microsoft.com/fwlink/?LinkId=272763)을 참조하세요.
  
## <a name="apply-rich-formatting-to-text-shapes-and-pictures"></a>텍스트, 도형 및 그림에 다양한 서식 적용
<a name="bkmk_Applyrichformatting"> </a>

PowerPoint Online에서는 사용할 수 없습니다. 추가적인 색상, 그라데이션, 스포이트, 효과 및 스타일 같은 고급 서식 기능은 PowerPoint 데스크톱 앱에서 사용할 수 있습니다. PowerPoint Online에서는 텍스트 상자를 추가하거나, 갤러리에서 도형을 선택하고 채우기/윤곽선/음영 효과를 정의하는 스타일을 적용할 수 있습니다. 
  
## <a name="apply-themes-and-theme-variants"></a>테마 및 테마 변형 적용
<a name="bkmk_ApplyThemes"> </a>

새 슬라이드를 프레젠테이션에 추가할 때 기본 제공 테마 및 변형을 갤러리에서 선택합니다. PowerPoint Online에서 지원되지 않는 테마는 프레젠테이션에 유지되지만, PowerPoint Online에서 테마를 수정할 수는 없습니다. 자세한 내용은 [브라우저와 PowerPoint 데스크톱 앱 간의 프레젠테이션 사용 차이점](https://go.microsoft.com/fwlink/?LinkId=272763)을 참조하세요.
  
## <a name="office-add-ins"></a>Office 추가 기능
<a name="bkmk_AppsOffice"> </a>

PowerPoint Online에서는 Office용 콘텐츠 추가 기능만 지원됩니다. 콘텐츠 추가 기능은 웹 기반 기능을 프레젠테이션과 일치하게 표시될 수 있는 콘텐츠로 통합합니다. 자세한 내용은 [Office용 추가 기능 유형](https://go.microsoft.com/fwlink/p/?LinkId=282310)을 참조하세요.
  
## <a name="arrange-objects"></a>개체 정렬
<a name="bkmk_ArrangeObjects"> </a>

PowerPoint Online에서 계층 또는 앞뒤로 도형이나 텍스트 상자를 이동하거나, 크기를 조정하거나, 회전 또는 정렬합니다. 도형을 개별적으로 작동하도록 그룹 해제합니다. 자세한 내용은 [브라우저와 PowerPoint 데스크톱 앱 간의 프레젠테이션 사용 차이점](https://go.microsoft.com/fwlink/?LinkId=272763)을 참조하세요.
  
## <a name="broadcast-slide-show"></a>슬라이드 쇼 브로드캐스트
<a name="bkmk_BroadcastSlideShow"> </a>

PowerPoint Online을 사용하여 슬라이드 쇼를 원격 대상에게 브로드캐스트하려면 PowerPoint 데스크톱 앱이 필요합니다. 원격 대상은 PowerPoint 데스크톱 앱 설치 여부에 상관없이 PowerPoint Online을 통해 라이브 프레젠테이션을 볼 수 있습니다. 자세한 내용은 [슬라이드 쇼 브로드캐스트](https://go.microsoft.com/fwlink/?LinkId=272765)를 참조하세요.
  
## <a name="clipboard"></a>클립보드
<a name="bkmk_Clipboard"> </a>

PowerPoint Online에서는 프레젠테이션의 콘텐츠를 잘라내고 복사하고 붙여넣을 수 있습니다. 사용자는 Office Online 프로그램 간에는 물론, Office Online와 컴퓨터에 설치된 Microsoft Office 데스크톱 앱 간에 텍스트를 복사하고 붙여넣을 수 있습니다. 자세한 내용은 [브라우저와 PowerPoint 데스크톱 앱 간의 프레젠테이션 사용 차이점](https://go.microsoft.com/fwlink/?LinkId=272763)을 참조하세요.
  
## <a name="create-and-manage-slides"></a>슬라이드 만들기 및 관리
<a name="bkmk_CreateManageSlides"> </a>

PowerPoint Online에서 슬라이드를 추가, 재정렬, 복제하고 숨기기 및 삭제합니다. 자세한 내용은 [브라우저와 PowerPoint 데스크톱 앱 간의 프레젠테이션 사용 차이점](https://go.microsoft.com/fwlink/?LinkId=272763)을 참조하세요.
  
## <a name="create-custom-animation"></a>사용자 지정 애니메이션 만들기
<a name="bkmk_Createcustomanimation"> </a>

PowerPoint Online에는 애니메이션 효과 갤러리가 포함되어 있습니다. PowerPoint Online에서 지원되지 않는 애니메이션은 프레젠테이션에 유지되고 슬라이드 쇼에 표시되지만, PowerPoint Online에서 수정할 수는 없습니다. 사용자 지정 애니메이션을 만들려면 PowerPoint 데스크톱 앱이 컴퓨터에 설치되어 있어야 합니다. 자세한 내용은 [PowerPoint 2013 데스크톱 앱으로 사용자 지정 애니메이션 만들기](https://go.microsoft.com/fwlink/p/?LinkId=282348)를 참조하세요.
  
## <a name="design-tools--advanced"></a>디자인 도구 - 고급
<a name="bkmk_DesigntoolsAdvanced"> </a>

PowerPoint Online에서는 사용할 수 없습니다. 슬라이드 마스터 만들기 및 레이아웃 수정 같은 고급 디자인 기능은 PowerPoint 데스크톱 앱에서만 사용할 수 있습니다. PowerPoint Online에서는 사용자 지정 템플릿 파일을 사용하여 새 파일을 만들거나, 미리 정의된 테마 목록에서 선택할 수 있습니다.
  
## <a name="dropbox"></a>Dropbox
<a name="bkmk_Dropbox"> </a>

Dropbox는 클라우드 저장소, 파일 동기화, 개인 클라우드 및 클라이언트 소프트웨어를 제공하는 파일 호스팅 서비스입니다.
  
## <a name="embed-presentation-on-web-or-blog-page"></a>웹 또는 블로그 페이지에 프레젠테이션 포함
<a name="bkmk_Embedpresentation"> </a>

PowerPoint Online에서는 Microsoft PowerPoint 데스크톱 앱이 없더라도 누구나 정보를 볼 수 있도록 프레젠테이션을 웹 사이트나 블로그에 포함할 수 있습니다. 포함된 뷰어에는 애니메이션, 전환 및 오디오/비디오가 고화질 읽기용 보기와 동일하게 표시됩니다. 자세한 내용은 [웹 또는 블로그 페이지에 프레젠테이션 포함](https://go.microsoft.com/fwlink/?LinkId=272766)을 참조하세요.
  
## <a name="font-formatting"></a>글꼴 서식
<a name="bkmk_FontFormatting"> </a>

PowerPoint Online을 텍스트에 굵게, 기울임꼴, 밑줄, 글꼴, 크기 및 색상을 적용할 수 있습니다. 또한 서식 복사를 사용하여 전체 도형의 서식을 복사를 사용할 수 있습니다.
  
## <a name="full-fidelity-reading-view"></a>고화질 읽기용 보기
<a name="bkmk_FullFidelityReading"> </a>

프레젠테이션의 애니메이션, 전환 및 오디오/비디오가 표시되어 만든 이가 의도한 대로 프레젠테이션을 볼 수 있습니다.
  
## <a name="full-ink-support"></a>전체 잉크 지원
<a name="bkmk_FullInkSupport"> </a>

PowerPoint Online에 잉크를 삽입할 수는 없지만, 해당 항목이 PowerPoint Online에 예상대로 표시됩니다.
  
## <a name="full-selection-of-animations-and-transitions"></a>애니메이션 및 전환의 전체 선택
<a name="bkmk_FullAnimationsTransitions"> </a>

PowerPoint Online에서는 8가지 전환 효과와 37가지 애니메이션을 지원합니다. 더 큰 범위를 선택하려면 PowerPoint 데스크톱 앱을 사용하세요. 
  
## <a name="headers-and-footers"></a>머리글 및 바닥글
<a name="bkmk_HeadersFooters"> </a>

PowerPoint Online에서는 날짜 및 슬라이드 번호를 포함한 머리글과 바닥글을 삽입, 편집하거나 삭제할 수는 없지만, 해당 항목이 PowerPoint Online에 예상대로 표시됩니다. 자세한 내용은 [브라우저와 PowerPoint 데스크톱 앱 간의 프레젠테이션 사용 차이점](https://go.microsoft.com/fwlink/?LinkId=272763)을 참조하세요.
  
## <a name="hyperlinks"></a>하이퍼링크
<a name="bkmk_Hyperlinks"> </a>

하이퍼링크를 삽입, 편집하고 하이퍼링크로 이동합니다. PowerPoint Online에서는 책갈피 링크가 작동하지만, 링크를 편집할 수는 없습니다.
  
## <a name="insert-online-video"></a>온라인 비디오 삽입
<a name="bkmk_InsertOnlineVideo"> </a>

비디오 및 오디오 콘텐츠는 읽기용 보기 및 슬라이드 쇼에서 재생되며 파일 크기를 100MB로 제한합니다. 온라인 비디오를 YouTube에서 삽입할 수 있고 PowerPoint Online에서는 미디어 컨트롤을 이동 및 삭제하고 크기를 조정할 수 있습니다. 그러나 YouTube 이외의 오디오 및 비디오를 삽입하려면 Microsoft PowerPoint 데스크톱 앱이 필요합니다.
  
## <a name="integration-with-excel-for-charts"></a>차트용 Excel과의 통합
<a name="bkmk_IntegrationExcel"> </a>

PowerPoint Online에서는 사용할 수 없습니다. PowerPoint Online에서 기존 프레젠테이션의 Excel 차트를 볼 수는 있지만, PowerPoint Online을 사용하여 프레젠테이션에서 Excel 차트를 편집하거나 삽입할 수는 없습니다. 자세한 내용은 [PowerPoint에 Excel 차트 복사](https://go.microsoft.com/fwlink/?LinkId=272773)를 참조하세요.
  
## <a name="navigation-slide-sorter"></a>탐색 - 슬라이드 분류기
<a name="bkmk_Navigation"> </a>

PowerPoint Online에서는 사용할 수 없습니다. PowerPoint Online에서는 슬라이드를 간편하게 정렬하고 구성할 수 있도록 슬라이드 보기를 미리 보기 형식으로 제공하는 슬라이드 분류기 보기를 사용할 수 없습니다. PowerPoint 데스크톱 앱에서 슬라이드 분류기 보기를 사용하여 슬라이드를 구성하고 섹션을 추가하고 슬라이드를 다른 범주로 정렬할 수 있습니다.
  
## <a name="offline-viewing-and-authoring"></a>오프라인으로 보기 및 작성
<a name="bkmk_Offlineviewingandauthoring"> </a>

PowerPoint Online은 웹 브라우저에서 시작되므로 인터넷에 연결해야 합니다. 오프라인으로 프레젠테이션에 액세스하려면 Microsoft PowerPoint 데스크톱 앱을 컴퓨터에 설치하고 이를 사용하여 슬라이드를 보고 편집해야 합니다.
  
## <a name="picture-cropping"></a>그림 자르기
<a name="bkmk_PictureCropping"> </a>

자르기 도구를 사용하여 그림의 피사체 프레이밍을 향상시킵니다. 그림 가장자리의 자르기 핸들 중 하나를 클릭하여 원하는 그림이 될 때까지 끌면 됩니다.
  
## <a name="pictures"></a>그림
<a name="bkmk_Pictures"> </a>

PowerPoint Online에서는 컴퓨터에 저장된 그림이나 Bing 이미지의 그림을 삽입할 수 있습니다. 그림을 이동하거나, 크기를 조정하고 자를 수 있으며 여러 그림 스타일을 적용할 수 있습니다. 효과를 적용하는 등의 더욱 정교한 그림 작업 기능은 PowerPoint Online에서 사용할 수 없습니다. PowerPoint Online에서는 스크린샷을 만들 수 없지만, 프레젠테이션에 있는 스크린샷이 PowerPoint Online에 그림으로 표시됩니다. 자세한 내용은 [브라우저와 PowerPoint 데스크톱 앱 간의 프레젠테이션 사용 차이점](https://go.microsoft.com/fwlink/?LinkId=272763)을 참조하세요.
  
## <a name="present-online-through-skype-for-business-or-the-office-presentation-service"></a>비즈니스용 Skype 또는 Office Presentation Service를 통해 온라인으로 프레젠테이션
<a name="bkmk_PresentOnline"> </a>

PowerPoint Online에서는 사용할 수 없습니다. Microsoft PowerPoint 데스크톱 앱을 통해 웹 브라우저에서 다른 사용자가 흐름을 따라올 수 있는 무료 공개 서비스인 Office Presentation Service를 사용하여 프레젠테이션을 전달할 수 있습니다. 자세한 내용은 [Office Presentation Service](https://go.microsoft.com/fwlink/?LinkId=272774)를 참조하세요.
  
## <a name="presenter-view"></a>발표자 보기
<a name="bkmk_PresenterView"> </a>

PowerPoint Online에서는 사용할 수 없습니다. Microsoft PowerPoint 데스크톱 앱에서만 발표자가 백그라운드에서 프레젠테이션 흐름, 메모, 주석 및 확대/축소 도수를 제어할 수 있습니다. PowerPoint Online에서는 이러한 기능이 지원되지 않습니다.
  
## <a name="print-to-pdf"></a>PDF로 인쇄
<a name="bkmk_PrinttoPDF"> </a>

PowerPoint Online에서는 슬라이드의 모든 레이아웃과 서식이 예상대로 인쇄되는 PDF 뷰어로 프레젠테이션을 인쇄할 수 있습니다. 자세한 내용은 [PowerPoint Online을 사용하여 수행할 수 있는 기본 작업](https://go.microsoft.com/fwlink/?LinkId=272775)을 참조하세요.
  
## <a name="proofing-tools"></a>교정 도구
<a name="bkmk_ProofingTools"> </a>

PowerPoint Online에서 기본 제공 사전을 사용하여 맞춤법을 검사하고 교정 언어를 설정할 수 있습니다. 그러나 PowerPoint Online에서는 사용자 지정 사전을 사용할 수 없고 번역이나 동의어를 포함하지 않습니다.
  
## <a name="real-time-co-authoring"></a>실시간 공동 작성
<a name="bkmk_Coauthoring"> </a>

여러 작성자가 PowerPoint Online 및 PowerPoint 2016에서 동시에 작업할 수 있습니다. 편집 시 충돌을 방지하고 변경 내용을 즉시 확인할 수 있도록 실시간 현재 상태를 통해 공동 작성자들이 프레젠테이션에서 작업 중인 위치를 확인할 수 있습니다. PowerPoint 2013에서는 동시 편집을 지원하지만 현재 상태가 표시되지 않으며 문서를 저장해야 여러 작성자가 변경 내용을 확인할 수 있습니다. PowerPoint의 실시간 공동 작성 기능에 대한 자세한 내용은 [PowerPoint 프레젠테이션에서 공동 작업](https://go.microsoft.com/fwlink/?linkid=844014)을 참조하세요.
  
## <a name="reviewer-tools---advanced"></a>검토자 도구-고급
<a name="bkmk_AdvancedReviewerTools"> </a>

PowerPoint Online에서는 사용할 수 없습니다. 충돌 병합 및 프레젠테이션 비교 같은 고급 검토자 기능은 PowerPoint 데스크톱 앱에서만 사용할 수 있습니다. PowerPoint Online에서는 설명을 보거나 추가 하거나 편집 하거나 삭제할 수 있습니다.
  
## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>권한 관리: IRM 및 암호 보호 적용 및 사용
<a name="bkmk_CreationPasswords"> </a>

PowerPoint Online에서는 사용할 수 없습니다. PowerPoint Online에는 IRM(정보 권한 관리)으로 보호되는 프레젠테이션이 표시됩니다. 그러나 브라우저에서 이 프레젠테이션을 편집할 수 없고, PowerPoint Online에서 IRM으로 보호된 프레젠테이션을 만들 수 없습니다. PowerPoint Online에서는 디지털 서명되거나 암호로 암호화된 프레젠테이션을 열 수 없습니다. 
  
## <a name="run-slide-show"></a>슬라이드 쇼 실행
<a name="bkmk_RunSlideShow"> </a>

PowerPoint Online에서는 웹 브라우저와 인터넷 연결을 통해서만 프레젠테이션을 실행할 수 있습니다. 자세한 내용은 [PowerPoint Online을 사용하여 수행할 수 있는 기본 작업](https://go.microsoft.com/fwlink/?LinkId=272775)을 참조하세요.
  
## <a name="save-as-or-download-a-copy"></a>복사본 다른 이름으로 저장 또는 다운로드
<a name="bkmk_Save"> </a>

PowerPoint Online에서는 변경할 때마다 작업을 저장합니다. 저장 명령은 없습니다. 복사본을 다운로드할 수 있지만 로컬 복사본을 편집하려면 Microsoft PowerPoint 데스크톱 앱이 있어야 합니다.
  
## <a name="shapes"></a>도형
<a name="bkmk_Shapes"> </a>

PowerPoint Online에 Word Art와 차트를 삽입할 수는 없지만, 해당 항목이 PowerPoint Online에 예상대로 표시됩니다. 편집용 보기에서 텍스트를 보고 편집하고 서식을 지정합니다. Word Art에 텍스트 효과를 적용하려면 Microsoft PowerPoint 데스크톱 앱이 필요합니다.
  
## <a name="share"></a>공유
<a name="bkmk_Share"> </a>

프레젠테이션을 SharePoint 문서 라이브러리에 저장하면 프레젠테이션이 온라인 상태이므로 전자 메일 첨부 파일이 아니라 링크를 보내서 프레젠테이션을 공유할 수 있습니다. 적절한 권한을 보유한 다른 사용자가 웹 브라우저나 모바일 장치에서 프레젠테이션을 볼 수 있습니다. 자세한 내용은 [프레젠테이션 공유](https://go.microsoft.com/fwlink/?LinkId=272776)를 참조하세요.
  
## <a name="slide-show"></a>슬라이드 쇼
<a name="bkmk_Slideshow"> </a>

PowerPoint Online에서는 슬라이드 쇼를 전체 화면 창으로 재생합니다. 스페이스바를 눌러서 슬라이드를 진행합니다. 슬라이드 애니메이션이 재생되지만 슬라이드 간에 페이드 및 닦아내기 전환만 지원됩니다. 자세한 내용은 [브라우저와 PowerPoint 데스크톱 앱 간의 프레젠테이션 사용 차이점](https://go.microsoft.com/fwlink/?LinkId=272763)을 참조하세요.
  
## <a name="smartart"></a>빠른
<a name="bkmk_SmartArt"> </a>

PowerPoint Online에서는 SmartArt를 삽입하고, 다른 레이아웃이나 색 구성표로 전환하며, SmartArt 스타일을 적용하고, 텍스트를 편집할 수 있습니다. 자세한 내용은 [브라우저와 PowerPoint 데스크톱 앱 간의 프레젠테이션 사용 시 차이점](https://go.microsoft.com/fwlink/?LinkId=272763)을 참조하세요.
  
## <a name="table-creationediting-and-formatting"></a>테이블 만들기/편집 및 서식 지정
<a name="bkmk_Tablecreation"> </a>

PowerPoint Online에서 테이블을 만들고 편집할 수 있으며 PowerPoint Online에서는 대부분의 테이블 기능을 지원합니다. 셀 병합 및 분할와 같은 고급 기능은 PowerPoint 데스크톱 앱을 사용합니다. 자세한 내용은 [브라우저와 PowerPoint 데스크톱 앱 간의 프레젠테이션 사용 차이점](https://go.microsoft.com/fwlink/?LinkId=272763)을 참조하세요.
  
## <a name="tell-me"></a>표시
<a name="Bkmk_TellMe_PPT"> </a>

PowerPoint Online에서 수행해야 하는 작업이 있지만 방법을 모를 때 표시 검색 기능을 사용하면 원하는 내용을 빠르게 찾을 수 있습니다. 표시 기능을 사용하면 수행하려는 작업을 파악하고 제안 사항을 활용하여 작업 시간을 단축할 수 있습니다.
  
## <a name="undo-and-redo"></a>실행 취소 및 다시 실행
<a name="bkmk_Undoredo"> </a>

활성 프레젠테이션에서 현재 편집 세션 중에 최근 작업을 개수 제한 없이 실행 취소(Ctrl + Z) 및 다시 실행(Ctrl + Y)합니다. 편집 세션 시간이 초과되거나 읽기용 보기로 30초 넘게 전환하면 실행 취소 기록이 원래대로 설정됩니다. 자세한 내용은 [PowerPoint Online의 바로 가기 키](https://go.microsoft.com/fwlink/?LinkId=272950)를 참조하세요.
  
## <a name="view-and-add-comments"></a>설명 보기 및 추가
<a name="bkmk_Viewaddcomments"> </a>

파일이 SharePoint Online 또는 비즈니스용 OneDrive에 저장 되어 있는 경우 설명을 추가, 편집 또는 삭제할 수 있습니다. 
  
파일이 OneDrive에 저장 된 경우 편집 보기에 있는 동안 설명을 추가, 편집 또는 삭제할 수 있으며, 읽기용 보기에서 설명을 보고 업데이트할 수 있습니다.
  
## <a name="view-and-edit-slide-notes"></a>슬라이드 메모 보기 및 편집
<a name="bkmk_Viewslidenotes"> </a>

PowerPoint Online에서 각 슬라이드에 대한 메모를 표시하거나 숨길 수 있습니다. 편집 모드에서 메모를 추가할 수도 있습니다.
  
## <a name="wysiwyg-viewing"></a>WYSIWYG 보기
<a name="bkmk_WYSIWYGViewing"> </a>

완성된 제품으로 인쇄하거나 표시할 때의 모양과 거의 흡사한 형태로 프레젠테이션을 편집합니다.
  
## <a name="feature-availability"></a>기능 가용성
<a name="bkmk_WYSIWYGViewing"> </a>

Office 365 계획, 독립 실행형 옵션 및 온-프레미스 솔루션별로 기능 가용성을 확인하려면 [Office Online 서비스 설명](office-online-service-description.md)을 참조하세요.
