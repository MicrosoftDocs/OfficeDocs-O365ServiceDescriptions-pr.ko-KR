# <a name="word-online"></a>Word Online

word Online (이전의 word Web App)을 사용 하면 Microsoft word 환경을 웹 브라우저로 확장 하 여 문서가 저장 된 웹 사이트에서 직접 문서로 작업할 수 있습니다. 
  
## <a name="alignment-guides-and-live-layout"></a>맞춤 안내서 및 라이브 레이아웃
<a name="bkmk_AlignmentGuidesLivelayout"> </a>

눈금자와 눈금선을 Word Online에서 사용할 수 없습니다. 자세한 내용은 [브라우저와 Word에서 문서 사용 시 차이점](https://go.microsoft.com/fwlink/p/?LinkId=271859)을 참조하세요.
  
## <a name="apply-styles"></a>스타일 적용
<a name="bkmk_Styles"> </a>

Word Online에서는 문서에 저장된 단락 및 문자 스타일을 텍스트에 적용할 수 있습니다. 자세한 내용은 [브라우저와 Word에서 문서 사용 시 차이점](https://go.microsoft.com/fwlink/p/?LinkId=271859)을 참조하세요.
  
## <a name="office-add-ins"></a>Office 추가 기능
<a name="bkmk_AppsforOffice"> </a>

Word용 추가 기능은 Office 스토어에서 다운로드할 수 있습니다. Office 추가 기능은 Word 데스크톱 앱에서만 작동하며, Word Online에서는 사용할 수 없습니다. 자세한 내용은 [Word용 추가 기능 가져오기](https://go.microsoft.com/fwlink/?LinkId=391443)를 참조하세요.
  
## <a name="autosave"></a>자동 저장
<a name="bkmk_Save"> </a>

문서를 변경하면 Word Online에서 문서를 자동으로 저장합니다.
  
## <a name="autotext"></a>상용구
<a name="bkmk_autotext"> </a>

상용구 갤러리에서 상용구를 사용하여 서식 설정된 텍스트의 블록을 삽입할 수 있습니다. 상용구는 텍스트의 동일한 큰 블록을 반복해서 입력해야 하고 텍스트에 많은 서식이 포함된 경우에 유용합니다. 선택한 텍스트를 상용구 갤러리에 추가하여 상용구 항목을 만듭니다. 자세한 내용은 [자동으로 텍스트 삽입](https://go.microsoft.com/fwlink/?linkid=846739)을 참조하세요.
  
## <a name="browse-and-start-from-professionally-designed-templates"></a>찾아보기 및 전문적으로 디자인된 템플릿에서 시작
<a name="bkmk_DesignedTemplates"> </a>

Word Online에서는 사용할 수 없습니다. Office.com에 호스트된 전문적으로 디자인된 템플릿을 대량으로 선택하여 문서를 시작하는 등의 고급 디자인 기능은 Word 데스크톱 앱에서만 사용할 수 있습니다. 그러나 Office 365 고객은 사용자가 새 문서를 만들 때 문서 라이브러리를 구성하여 사용자 지정 템플릿을 시작할 수 있습니다. 자세한 내용은 [목록 또는 라이브러리에 콘텐츠 형식 추가](https://go.microsoft.com/fwlink/p/?LinkId=391446)를 참조하세요.
  
## <a name="bullets-and-numbering"></a>글머리 기호 및 번호 매기기
<a name="bkmk_BulletsNumbering"> </a>

Word Online에서는 글머리 기호 스타일 세 가지 또는 번호 매기기 스타일 다섯 가지 중에서 선택해 적용할 수 있습니다. 들여쓰기 및 내어쓰기 단추를 클릭하여 문서에서 기존 글머리 기호 및 번호 목록과 Word 데스크톱 앱에서 만들어진 목록의 목록 수준을 변경합니다. 자세한 내용은 [브라우저와 Word에서 문서 사용 시 차이점](https://go.microsoft.com/fwlink/p/?LinkId=271859)을 참조하세요.
  
## <a name="captions"></a>캡션
<a name="bkmk_Captions"> </a>

Word Online에서는 사용할 수 없습니다. Word 데스크톱 앱이 컴퓨터에 설치된 경우 Word를 사용하여 문서에 캡션을 추가할 수 있습니다. 자세한 내용은 [Word 2013에서 캡션 추가](https://go.microsoft.com/fwlink/p/?LinkId=282312)를 참조하세요.
  
## <a name="citations-and-bibliography"></a>인용 및 참고 문헌
<a name="bkmk_Citations"> </a>

Word Online에서는 사용할 수 없습니다. Word Online에서 기존 인용을 볼 수는 있지만 인용을 추가할 수는 없습니다. Word 데스크톱 앱이 컴퓨터에 설치된 경우 Word를 사용하여 참고 문헌 및 인용을 문서에 추가할 수 있습니다. 자세한 내용은 [Word 2013에서 참고 문헌 만들기](https://go.microsoft.com/fwlink/p/?LinkId=282314)를 참조하세요. 
  
## <a name="clipboard"></a>클립보드
<a name="bkmk_Clipboard"> </a>

웹 브라우저의 기술적 제한 때문에 Office Online에서 텍스트를 복사하고 붙여넣는 방법은 Office 데스크톱 응용 프로그램에서 텍스트를 복사하고 붙여넣는 방법과 다릅니다. 자세한 내용은 [Office Online에서 복사 및 붙여넣기](https://go.microsoft.com/fwlink/p/?LinkId=282321)를 참조하세요.
  
## <a name="collaboration--advanced"></a>공동 작업 - 고급
<a name="bkmk_AdvancedCollaboration"> </a>

Word Online에서는 사용할 수 없습니다. 변경 내용 켜기, 문서 병합, 비교 및 결합 같은 고급 공동 작업 기능은 Word 데스크톱 앱에서만 사용할 수 있습니다. 예상대로 추적한 변경 사항은 보기 모드에서 사용 중일 때 Word Online에 표시됩니다.
  
## <a name="content-controls"></a>콘텐츠 컨트롤
<a name="bkmk_contentcontrols"> </a>

콘텐츠 컨트롤은 템플릿, 양식 및 문서에서 사용하도록 추가하고 사용자 지정할 수 있는 개별 컨트롤입니다. 콘텐츠 컨트롤에 대한 자세한 내용은 [콘텐츠 컨트롤](https://go.microsoft.com/fwlink/p/?LinkId=846741)을 참조하세요.
  
## <a name="cover-pages"></a>표지
<a name="bkmk_coverpage"> </a>

Microsoft Word는 미리 설계된 편리한 표지의 갤러리를 제공합니다. 간편하게 표지를 선택하고 샘플 텍스트를 직접 바꿀 수 있습니다.
  
## <a name="creation-of-advanced-document-elements"></a>고급 문서 요소 만들기
<a name="bkmk_Creationadvanceddocumentelements"> </a>

Word Online에서는 사용할 수 없습니다. 상용구, 표지, 참고 문헌, 목차, 인덱스, 수식, 워터마크 같은 고급 문서 요소는 Word 데스크톱 앱에서만 사용할 수 있습니다. 
  
## <a name="document-parts-and-themes"></a>문서 요소 및 테마
<a name="bkmk_Documentpartsandthemes"> </a>

Word Online에서는 사용할 수 없습니다. 콘텐츠 제어, 미리 서식이 지정된 색 구성표 모음, 글꼴 구성표 및 그래픽 효과 같은 문서 요소와 테마는 Word 데스크톱 앱에서만 사용할 수 있습니다. 
  
## <a name="document-translation"></a>문서 번역
<a name="bkmk_DocumentTranslation"> </a>

문서를 브라우저에서 직접 번역합니다. 이 기능은 문서 주석까지도 번역해 줍니다.
  
## <a name="dropbox"></a>Dropbox
<a name="bkmk_Dropbox"> </a>

Dropbox는 클라우드 저장소, 파일 동기화, 개인 클라우드 및 클라이언트 소프트웨어를 제공하는 파일 호스팅 서비스입니다.
  
## <a name="equations"></a>수식
<a name="bkmk_equations"> </a>

Microsoft Word에는 문서에 적용할 수 있는 수식뿐만 아니라 쓰기 및 편집 수식에 대한 기본 제공 지원도 포함되어 있습니다.
  
## <a name="find-and-replace"></a>찾기 및 바꾸기
<a name="bkmk_Find"> </a>

Word Online과 Word 데스크톱 앱에서 모두 찾기 및 바꾸기 도구를 사용하여 문서 텍스트를 찾고 바꿀 수 있습니다. Word Online에서 찾기 및 바꾸기를 사용하는 방법에 대한 자세한 내용은 [Word Online의 새로운 기능](https://go.microsoft.com/fwlink/?LinkId=389520)을 참조하세요.
  
## <a name="font-formatting"></a>글꼴 서식
<a name="bkmk_Fontformatting"> </a>

Word Online에서는 굵게, 기울임꼴, 밑줄, 위 첨자 등의 글꼴과 글꼴 크기 및 다양한 글꼴 서식 특성을 적용할 수 있습니다. 텍스트를 강조 표시하고 글꼴 색을 변경하고 선택한 텍스트의 서식을 지웁니다. 문서 테마 서식을 지정하면 Word Online에서 텍스트 서식을 지정할 수 있습니다. 글꼴 갤러리의 상단에서 활성 테마 글꼴을 찾고 글꼴 색 갤러리에서 테마 색을 찾습니다. 자세한 내용은 [브라우저와 Word에서 문서 사용 시 차이점](https://go.microsoft.com/fwlink/p/?LinkId=271859)을 참조하세요.
  
## <a name="footnotes-and-endnotes"></a>각주와 미주
<a name="bkmk_Footnotes"> </a>

Word Online에서는 문서에 각주와 미주를 추가할 수 있습니다. 자세한 내용은 [각주 및 미주 추가](https://go.microsoft.com/fwlink/p/?LinkId=397306)를 참조하세요.
  
## <a name="full-fidelity-reading-view"></a>고화질 읽기용 보기
<a name="bkmk_FullFidelityReadingView"> </a>

OneDrive 또는 SharePoint에서 문서를 열면 Word Online의 문서가 브라우저에서 열립니다. Word 데스크톱 앱에서 문서를 인쇄 모양 보기로 열면 레이아웃과 서식이 표시됩니다.
  
## <a name="graphic-effect-presets"></a>그래픽 효과 미리 설정
<a name="bkmk_presets"> </a>

미리 설정을 사용하여 회전, 그림자, 채도 및 텍스트 효과 등의 특수 효과를 그래픽에 적용할 수 있습니다.
  
## <a name="headers-and-footers"></a>머리글 및 바닥글
<a name="bkmk_WordHeadersandFooters"> </a>

Word Online과 Word 데스크톱 앱에서 모두 문서에 머리글과 바닥글을 삽입할 수 있습니다. 또한 문서 첫 페이지를 제외한 모든 페이지에 머리글과 바닥글을 적용할 수도 있습니다. Word Online에서 머리글과 바닥글을 사용하는 방법에 대한 자세한 내용은 [Word Online의 새로운 기능](https://go.microsoft.com/fwlink/?LinkId=389520)을 참조하세요.
  
## <a name="hyperlinks"></a>하이퍼링크
<a name="bkmk_Hyperlinks"> </a>

Word Online에서는 문서 내에 책갈피 링크를 손쉽게 만들고, 커서가 잉크를 가리킬 때 표시되는 스크린 팁을 설정할 수 있습니다. 자세한 내용은 [Word Online의 하이퍼링크](https://go.microsoft.com/fwlink/p/?LinkId=271863)를 참조하세요.
  
## <a name="index"></a>색인
<a name="bkmk_index"> </a>

인덱스는 문서에 언급된 용어 및 주제와 함께 해당 내용이 표시된 페이지를 나열합니다. 인덱스를 만들려면 문서의 기본 항목 이름과 상호 참조를 제공하여 인덱스 항목을 표시한 다음 인덱스를 작성합니다.
  
## <a name="ink-editor"></a>잉크 편집기
<a name="BKMK_InkEditor"> </a>

잉크 편집기에서는도로를 사용 하 여 텍스트를 선택 하 고 텍스트를 삭제할 수 있는 기본 제스처를 통해 문서를 편집 하 고 서식을 지정할 수 있습니다.
  
## <a name="insert-online-pictures"></a>온라인 그림 삽입
<a name="bkmk_InsertPicturessClipArt"> </a>

Word Online에서는 로컬 하드 드라이브에 저장된 파일 또는 Bing 이미지의 그림을 삽입할 수 있습니다. 자세한 내용은 [브라우저와 Word에서 문서 사용 시 차이점](https://go.microsoft.com/fwlink/p/?LinkId=271859)을 참조하세요.
  
## <a name="irm-and-password-protection"></a>IRM 및 암호 보호
<a name="bkmk_IRMpasswoprdprotect"> </a>

Word Online에는 IRM(정보 권한 관리)으로 보호되는 문서가 표시됩니다. 그러나 브라우저에서는 이러한 문서를 편집할 수 없고, Word Online에서는 IRM으로 보호된 문서를 만들 수 없습니다. Word Online에서는 암호로 암호화된 문서를 열 수 없습니다. IRM으로 보호된 문서 만들기, 암호 보호 적용 같은 고급 문서 보호 기능은 Word 데스크톱 앱에서만 사용할 수 있습니다.
  
## <a name="learning-tools"></a>학습 도구
<a name="BKMK_LearningTools"> </a>

학습 도구는 단어를 올바르게 발음 하 고, 빠르고 정확 하 게 읽을 수 있으며, 읽은 내용을 이해 하는 데 도움이 되도록 디자인 되었습니다. 
  
## <a name="lists"></a>목록
<a name="BKMK_Lists_WordOnline"> </a>

Word Online에서는 기존의 번호 매기기 목록 바로 아래에 입력을 시작하면 다음 줄도 자동으로 해당 목록에 포함됩니다. 자세한 내용은 [Word Web App에서 목록을 만들 수 있나요?](https://go.microsoft.com/fwlink/p/?LinkId=397307)를 참조하세요.
  
## <a name="mail-merge"></a>메일 병합
<a name="bkmk_Mailmerge"> </a>

Word Online에서는 사용할 수 없습니다. Excel 스프레드시트를 통한 메일 병합 같은 고급 데이터 통합 기능은 Word 데스크톱 앱에서만 사용할 수 있습니다. 
  
## <a name="offline-viewing-and-authoring"></a>오프라인으로 보기 및 작성
<a name="bkmk_Offlineviewingauthoring"> </a>

Word Online에서는 사용할 수 없습니다. Office Online을 사용하려면 인터넷에 연결해야 하며 인터넷 브라우저가 필요합니다. 조직의 온-프레미스 Office Web Apps 서버나 인터넷에서 연결이 끊어진 동안 문서를 보고 편집하려면 Word 데스크톱 앱을 컴퓨터에 설치해야 합니다. Word 데스크톱 앱이 설치되어 있으면 Word의 전체 기능을 사용하여 문서를 보고 편집할 수 있습니다. 문서를 저장하면 온라인 문서에 변경 사항이 업데이트됩니다. 또한 문서를 다운로드하고 오프라인 문서 복사본으로 저장할 수 있지만 온라인 버전과 동기화되지는 않습니다. 자세한 내용은 [Office Web Apps 서버](https://go.microsoft.com/fwlink/p/?LinkId=282318)를 참조하세요.
  
## <a name="page-breaks"></a>페이지 나누기
<a name="bkmk_PageBreaks"> </a>

Word Online과 Word 데스크톱 앱에서 모두 페이지 나누기를 삽입할 수 있습니다.
  
## <a name="page-layout-tools"></a>페이지 레이아웃 도구
<a name="bkmk_PageLayoutTools"> </a>
